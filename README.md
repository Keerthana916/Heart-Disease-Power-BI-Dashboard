# Heart Disease Analytics Dashboard

Interactive Power BI dashboard built to analyze heart disease patient data.

## Features
- KPI Cards
- Interactive slicers
- Disease distribution analysis
- Cholesterol analysis
- Heart rate trends
- Gender distribution

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Preview
![Dashboard](https://github.com/Keerthana916/Heart-Disease-Power-BI-Dashboard/blob/main/Heart-Disease-Dashboard-Photo.png)

## Insights
- Majority of patients are male
- Higher cholesterol observed among disease patients
- Heart disease cases mostly appear in middle-aged groups
